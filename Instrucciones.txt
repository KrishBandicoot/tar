Para usar tienes que abrir XAMPP e iniciar apache y MySQL. Después abres MySQL y abres una base de datos en el puerto 3306 con el nombre basedatos.
finalmente vuelves  a visual code y vasa  la pestaña de springboot para iniciar el backend.
Testeado con swagger.
